# homebrew-rediseen
Homebrew Formula for Rediseen

Local dev process:

```bash
#Update 
vi /usr/local/Homebrew/Library/Taps/xd-deng/homebrew-rediseen/rediseen.rb

brew install --build-from-source xd-deng/rediseen/rediseen
brew test xd-deng/rediseen/rediseen
```
